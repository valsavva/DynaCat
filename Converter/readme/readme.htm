см "удаляем первичный слой.png"

======================================

АТРИБУТЫ (как для Tower)

- просто вписываем в свойства объекта
см "собственные не графические атрибуты объекта.png"

======================================

АТРИБУТЫ ДЕТЕЙ (как для Teleport)

- вписываем в свойства объекта атрибут, указав имя ребенка, заключенное в двойное нижнее подчеркивание, типа __Teleport__Id="funkyShit" это значит что атрибут предназначен для элемента Teleport, а не для блока в котором мы вписываем в свойствах эту строку.
см "атрибуты детей объекта.png"

======================================

ПРАВИЛА И ОГРАНИЧЕНИЯ ПРИ СОЗДАНИИ УРОВНЯ

- при импорте в уровень картинки или объекта (напр. путем drag n drop) необходимо выбирать link (не embed!)
- картинки/объекты нельзя вращать и отражать, но можно произвольно менять размеры
- также в процессе редактирования нельзя менять ориентацию листа (портрет/альбом) т.е. вращать слой. (это можно только в самом начале, когда нет еще слоев)
- нельзя группировать объекты
